Weighted Voronoi Stippling dient der Verbesserung von Konturen und Schattierungen von Abbildungen, welche mittels Stippling erstellt wurden.

Zur Verwendung des Repositorys muss das gewählte Bild in Input eingefügt werden und im R Code dann der entsprechende Name angepasst werden.

Wurde der Code durchgeführt, so befindet sich dann das Ergebnis im Output Ordner.
